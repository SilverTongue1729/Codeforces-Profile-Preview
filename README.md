# Codeforces Profile Preview Extension

A Chrome extension that shows a preview of a Codeforces user profile when hovering over their username link.

## Features

- **Instant Preview**: Hover over any Codeforces username to see a summary card.
- **Key Stats**: Displays rank, rating, max rating, contribution, friend count, and last visit.
- **Friend Toggle**: Add or remove friends directly from the preview card.
- **Blog Count**: Shows the number of blog entries.
- **Smart Positioning**: The preview automatically positions itself to stay within the viewport.
- **Rate Limit Handling**: Gracefully handles API rate limits.
- **Custom Styling**: Matches Codeforces' native look and feel.

## Installation

1.  Download the latest release zip file.
2.  Open Chrome and go to `chrome://extensions`.
3.  Enable "Developer mode" in the top right.
4.  Drag and drop the zip file (or unzip and click "Load unpacked") to install.

## Development

1.  Clone this repository.
2.  Make your changes in `content.js` or `styles.css`.
3.  Reload the extension in `chrome://extensions` to test your changes.
